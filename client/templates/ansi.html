About ANSI Codes

The output code represents your pixel art in ANSI format:

  • B - Background (empty)
  • O - Orange pixel
  • w - White pixel
  • B - Black pixel
  • n - New line

You can copy this code and use it in terminal applications or other systems that support ANSI formatting.
